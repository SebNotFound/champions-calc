# Sample Team Preview screenshots

Drop real Pokémon Champions **Team Preview** screenshots here. They're used to
build and tune the free on-device recognizer (detecting the blue/red panels,
cropping each slot, and matching the enemy sprites).

Suggested names (PNG or JPG both fine):

* `team-preview-1.png`
* `team-preview-2.png`

These are local test fixtures. If you'd rather not commit game screenshots to
git, tell me and I'll add `samples/*.png` to `.gitignore`.
